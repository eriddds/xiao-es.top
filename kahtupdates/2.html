Types of Computer Viruses

There are multiple types of computer viruses, including but not limited to:

  • File Viruses
    • Parasitic Viruses
      • Characteristics: Attach to executable files and modify the file's code structure.
      • Common Forms: Executable files (e.g., .exe, .com).
      • File Extensions: .exe, .com.
    • Overwriting Viruses
      • Characteristics: Overwrite or replace the content of target files, causing the original files to be lost or damaged.
      • Common Forms: Executable files or data files.
      • File Extensions: .exe, .com, .dat.
    • Companion Viruses
      • Characteristics: Create files with the same name as the target file but with a different extension; when the user runs the target file, the virus file is actually executed.
      • Common Forms: Companion files to executable files.
      • File Extensions: .bat, .cmd.
  • Boot Sector Viruses
    • Master Boot Record Viruses
      • Characteristics: Infect the Master Boot Record (MBR) of the hard drive or the boot sector of a floppy disk, and execute when the system starts.
      • Common Forms: Boot sector of the system at startup.
      • File Extensions: No specific file extension, affects the MBR of the hard drive.
    • Partition Table Viruses
      • Characteristics: Modify the partition table of the hard drive, causing the system to fail to start normally or files to be lost.
      • Common Forms: Partition table structure.
      • File Extensions: No specific file extension, affects the partition table of the hard drive.
  • Macro Viruses
    • Document Macro Viruses
      • Characteristics: Utilize the macro functions in Office documents; when users open documents containing malicious macros, the virus is triggered.
      • Common Forms: Office documents.
      • File Extensions: .doc, .xls, .ppt.
    • Template Macro Viruses
      • Characteristics: Infect Office template files; when users create new documents, the virus automatically spreads to the new documents.
      • Common Forms: Office template files.
      • File Extensions: .dot, .xlt, .pot.
  • Worm Viruses
    • Network Worms
      • Characteristics: Spread automatically through network vulnerabilities without user intervention, quickly infecting a large number of systems.
      • Common Forms: Network data packets or vulnerability exploitation code.
      • File Extensions: No specific file extension, typically exist as network data packets.
    • Email Worms
      • Characteristics: Spread through email attachments or links; when users open attachments or click on links, the virus is activated.
      • Common Forms: Email attachments.
      • File Extensions: .exe, .scr, .bat.
  • Script Viruses
    • Batch Script Viruses
      • Characteristics: Utilize Windows batch scripts to perform operations such as file deletion and system settings modification.
      • Common Forms: Batch script files.
      • File Extensions: .bat, .cmd.
    • JavaScript Viruses
      • Characteristics: Utilize JavaScript scripts, typically spread through web pages or email attachments.
      • Common Forms: Web scripts or email attachments.
      • File Extensions: .js, .html.
  • Folder Viruses
    • Folder Icon Disguise Viruses
      • Characteristics: Disguise virus files as folder icons; when users double-click the "folder," they are actually executing the virus file.
      • Common Forms: Executable files disguised as folders.
      • File Extensions: .exe, .scr.
  • Logic Bombs
    • Condition-triggered Logic Bombs
      • Characteristics: Execute malicious operations when specific conditions are met (e.g., a particular date, a specific file is opened).
      • Common Forms: Malicious code embedded in normal programs or scripts.
      • File Extensions: No specific file extension, typically hidden in executable files or script files.
  • Backdoor Viruses
    • Remote Access Backdoors
      • Characteristics: Provide attackers with remote access to the infected system, allowing them to control the system remotely.
      • Common Forms: Backdoor programs or services.
      • File Extensions: No specific file extension, typically exist as executable files.
  • File Encryption Viruses
    • Ransomware
      • Characteristics: Encrypt users' files and demand a ransom for the decryption key.
      • Common Forms: Ransomware programs.
      • File Extensions: No specific file extension, typically exist as executable files.
  • Polymorphic Viruses
    • Polymorphic Engine Viruses
      • Characteristics: Change their code each time they infect a file, making virus detection more difficult.
      • Common Forms: Executable files.
      • File Extensions: .exe, .com.
  • Stealth Viruses
    • Rootkit Viruses
      • Characteristics: Hide themselves and their malicious activities, making it difficult for users and security software to detect their presence.
      • Common Forms: System processes or services.
      • File Extensions: No specific file extension, typically hidden in system files.

# If your computer has been infected with a virus, you can follow these steps to resolve the issue:

  1. Disconnect from the network

    Immediately disconnect your computer from the network to prevent the virus from spreading further over the network or downloading more malicious software.

  2. Boot into Safe Mode

    Restart your computer and boot into Safe Mode. Safe Mode loads the minimum system files and drivers, which helps prevent the virus from running during startup.

  3. Run Antivirus Software

    Use antivirus software to perform a full scan and remove the virus. If you don't have antivirus software, you can download and install one from a trusted source in Safe Mode.

  4. Delete Suspect Files

    Check for suspicious files and programs in your system and delete them. Pay special attention to unknown sources or suspicious executable files.

  5. Update the System and Software

    Ensure that your operating system and all software are updated to the latest version to patch any existing security vulnerabilities.

  6. Restore the System

    If the problem is severe, consider using the system restore feature to restore to an earlier system restore point or reinstall the operating system.

  7. Backup Important Data

    After resolving the issue, ensure to back up all important data to prevent data loss in the future.

By following these steps, you can effectively resolve the issue of your computer being infected with a virus. If the problem persists, it is recommended to seek professional technical support.


# How to Identify Virus Files

Identifying virus files is an important step in protecting your computer's security. Here are some common methods to identify virus files:

  1. Check File Name and Path

    Virus files often use the names of normal files or system files to disguise themselves, such as changing svchost.exe to svch0st.exe, or appearing in folders that they shouldn't be in, such as the windows folder where you find files like RAV or RealPlayer.

  2. Check File Properties

    Check the file's version information and manufacturer information. Normal files usually have detailed version information and manufacturer information, while virus files may not have version information, or the manufacturer information may not match the file name, such as a strange file name but Microsoft's manufacturer information.

  3. Scan with Antivirus Software

    Use antivirus software to scan suspicious files. Antivirus software will use its virus database, including feature codes, behavior patterns, etc., to determine whether a file is a virus.

  4. Analyze File Behavior

    Run suspicious files in a sandbox environment and observe their behavior. Virus files usually perform some abnormal operations, such as frequently accessing the registry, modifying system files, downloading other malicious files, etc.

  5. Check File MD5 Value

    Calculate the MD5 value of the file and compare it with the MD5 values in the virus database. If a match is found, it indicates that the file is a malicious file.

  6. Check the API Called by the File

    Use disassembly tools to view the API called by the file. Virus files may call some APIs that are unrelated to normal programs, such as frequently calling CreateFile, WriteFile, etc. file operation APIs.
