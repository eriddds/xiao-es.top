测试工具

  • afrog - 一款快速、稳定的高性能漏洞扫描器。 使用文档前往官网
  • xray - 一款完善的安全评估工具，支持常见 web 安全问题扫描。 使用文档前往官网
  • nuclei - 使用零误报的定制模板对主机进行批量快速扫描。 使用文档前往官网
  • pocsuite3 - 由知道创宇404团队开发的开源远程漏洞测试框架。 使用文档前往官网
  • POC-bomber - 利用大量高威胁poc/exp快速获取目标权限，用于渗透和红队快速打点。 使用文档前往官网
  • GitHack - .git泄露利用测试脚本，通过泄露的文件，还原重建工程源代码。 使用文档前往官网
  • IIS_shortname_Scanner - 利用短文件名漏洞进行文件探测的扫描器。 使用文档前往官网
  • Cr3dOv3r - 一款开源的密码泄露数据收集工具，支持从多个数据泄露源收集泄露的用户名和密码。 使用文档前往官网
  • Metasploit - 渗透测试框架，拥有庞大的编程员爱好者群体，可以测试众多操作系统和应用程序中存在的安全漏洞。 使用文档前往官网
  • Nmap - 网络发现和攻击界面测绘的首选工具，用于主机发现、端口扫描、操作系统检测等。 使用文档前往官网
  • Burp Suite - 一款应用测试工具，适合核心评估。 使用文档前往官网
  • OWASP ZAP - 与Burp Suite相提并论的应用测试工具，适合应用安全新手。 使用文档前往官网
  • SQLmap - 自动化SQL注入和数据库接管工具，支持所有常见数据库平台。 使用文档前往官网
  • Cobalt Strike - 渗透测试工具，集成了端口转发、服务扫描，自动化溢出等功能。 使用文档前往官网
  • CME(CrackMapExec) - 后漏洞利用工具，可帮助自动化大型活动目录(AD)网络安全评估任务。 使用文档前往官网
  • Impacket - Python类库，用于对SMB1-3或 IPv4 / IPv6 上的TCP、UDP、ICMP、IGMP和ARP之类的协议进行低级编程访问。 使用文档前往官网
  • PowerSploit - 一系列模块的集合，用于Windows上的PowerShell，功能包括驻留、杀软规避、渗漏、代码执行等。 使用文档前往官网
  • BeEF(浏览器漏洞利用框架) - 利用客户端攻击方法，评估目标环境切实安全状态的工具。 使用文档前往官网
  • THC-Hydra - 网络登录破解器，支持多个服务。 使用文档前往官网
  • 社会工程工具箱(SET) - 面向社会工程的渗透测试框架。 使用文档前往官网
  • Aircrack-ng - 无线渗透测试领域的重量级选手，提供了一整套工具和功能，专注于Wi-Fi安全的各个领域。 使用文档前往官网
  • fscan - 一款用于内网扫描和侦察的工具。 使用文档前往官网
  • Godzilla - 一款用于Web应用渗透测试的工具。 使用文档前往官网
  • LaZagne - 一款用于收集密码和凭证的工具。 使用文档前往官网
